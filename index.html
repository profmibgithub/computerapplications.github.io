Semester 1 - Foundation Semester

Build fundamental understanding of computer systems, programming concepts, and digital technologies.

Major Course

CAP122J - Computer Fundamentals 4+2 Credits
Learning Outcomes:
  • Understand computer system components and architecture
  • Master number systems and binary operations
  • Differentiate between software types and programming paradigms
  • Apply internet technologies for communication and web services
Unit-wise Syllabus:

  • Computer History and Evolution
  • Generations of Computers
  • Memory Hierarchy (Primary, Secondary, Cache)
  • Input/Output Devices
  • Number Systems (Binary, Octal, Hexadecimal)
  • Binary Operations and Conversions

  • System Software vs Application Software
  • Programming Languages (Low-level, High-level)
  • Compilers vs Interpreters vs Assemblers
  • Operating System Functions
  • Software Development Life Cycle

  • Data Communication Modes
  • Network Types (LAN, WAN, MAN)
  • Network Topologies
  • Communication Protocols (TCP/IP, HTTP)
  • Network Components (Hub, Switch, Router)

  • World Wide Web (WWW)
  • Web Browsers and Search Engines
  • URLs and Domain Names
  • Email Systems and Services
  • E-governance and Digital India
  • Web Hosting and Cloud Services
Lab Practicals:
  • MS Word: Document formatting, tables, mail merge
  • MS Excel: Formulas, charts, pivot tables
  • MS PowerPoint: Presentations, animations, multimedia
  • Internet Usage: Browsing, downloading, online services
  • Email Setup: Creating accounts, sending attachments
Textbooks & References:
  • Fundamentals of Computers - V Rajaraman
  • Computer Today - Donald H. Sanders
  • Introduction to Computers - Peter Norton
Assessment Methods:
  • Mid-term Exam: 30%
  • End-term Exam: 50%
  • Lab Assessment: 20%

Minor Course

ACP124N - Digital Electronics 4+2 Credits

Fundamentals of digital systems, boolean algebra, logic gates, and microprocessor basics.

Topics Covered:
  • Number Systems and Binary Codes
  • Boolean Algebra and Logic Simplification
  • Logic Gates (AND, OR, NOT, NAND, NOR, XOR)
  • Combinational Circuits (Adders, Multiplexers, Decoders)
  • Sequential Circuits (Flip-flops, Counters, Registers)
  • Introduction to Microprocessors (8085/8086)

Skill Courses

JSD122S - Programming with C (Basic) 2+2 Credits

Track: Junior Software Developer

WDP122S - Internet Basics and HTML 2+2 Credits

Track: Web Developer
